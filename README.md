# Dockerized Vite-UI-App

Containerized Vite UI application using Docker.

## Original Project
Based on: [https://github.com/agmmnn/tauri-ui.git]

## My Work
- Created Dockerfile with Multistage build for containerization
- Optimized with Alpine base image
- Published to Docker Hub

## Quick Start
```bash
docker pull farhan249/vite-ui-app
docker run -p 8000:80 farhan249/vite-ui-app
```

Open http://localhost:8000
